## Converted Harness Dataset

- Output directory: data/cell_lines/k562
- Format: split NPZ files plus manifest.json
- Perturbations: 105
- Feature dimension: 105 (perturbation one-hot)
- Target genes: 1000
- Splits: {'train': 73, 'val': 16, 'test': 16}
- Label counts: {'0': 5250, '1': 94500, '2': 5250}
- Label rule: per perturbation, delta vs control mean expression; <=5th percentile down, >=95th percentile up, otherwise unchanged

The data directory is intentionally ignored by git.

## Interpretation And Limitations

This is a real-data harness demo, not a full paper reproduction.

What is real:

- The source is a real K562 Perturb-seq dataset from Norman/Weissman 2019 via scPerturb.
- The train/val/test NPZ files contain real perturbation-level expression-derived labels.
- The same training and MCTS loop used by the synthetic demo runs on this real cell-line dataset.

What is still simplified:

- The input `features` are perturbation one-hot vectors, not AIDO/ESM2/scFoundation/STRING embeddings.
- DEG labels are generated by a simple per-perturbation delta-vs-control percentile rule.
- The target gene set is a top-1000 expressed-gene subset, not yet aligned to the paper's exact benchmark protocol.
- The MCTS run is intentionally short and demonstrates mechanics rather than final performance.
